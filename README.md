# Vinland Project

Vinland its an ecommerce created by Camilo, Cristian, Stiven and Valen, where you can buy your own vinyls with the bests prices ever

## Dependencies

This project was created using `npx create-react-app`

- react-router-dom
- fontawesome
- react-redux
- bootstrap
